# Sanawer-Batool Portfolio


### Prerequisites

* [Node.js](https://nodejs.org/)
* [Git](https://git-scm.com/)

### Clone and Run

```bash
git clone https://github.com/Sanawer-Batool/Sanawer-Batool.github.io.git
cd Sanawer-Batool.github.io
npm install
npm run dev
```

The app will be available at: [http://localhost:5173](http://localhost:5173)

---

## ☁️ Deployment

### Deploy on Vercel

1. Push the code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your repository
4. Click **Deploy**

The live website will be hosted on a custom subdomain (e.g. `https://your-name.vercel.app`)
